# 🔐 Generador de Contraseñas Seguras

Este proyecto es una herramienta para **generar contraseñas aleatorias** con distintos niveles de seguridad. Se ofrece una interfaz de **línea de comandos (CLI)** y una **interfaz gráfica (GUI)** para facilitar su uso.

## 📌 Características
✅ Genera contraseñas seguras con diferentes niveles de complejidad.  
✅ Guarda automáticamente las contraseñas en un archivo (`passwords.txt`).  
✅ Interfaz gráfica intuitiva para generar contraseñas fácilmente.  
✅ Código modular y estructurado profesionalmente.
